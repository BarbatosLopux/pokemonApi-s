# POKEMON API 
A partir de este proyecto podemos identificar el consumo de api's, en este caso contando con la informacion de Pockeapi, recreando un apartado parecido a la "PokeDex" para ver informacion de Pokemones.

## TECNOLOGIAS USADAS 
* HTML
* CSS 
* JAVASCRIPT 
* POCKEAPI 
* MOCKAPI
